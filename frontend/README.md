# 💻 Frontend – Smart Traffic Management System

This is the **React-based frontend** for the Smart Traffic Management System.  
It provides a clean and interactive user interface to monitor live traffic, view real-time camera streams, and visualize traffic conditions at different junctions.

The frontend communicates with the Flask backend to fetch live data and display it in an easy-to-understand format, enabling efficient traffic monitoring and control.

---

## 🚀 Key Features

- 🎥 Live traffic video streaming  
- 📊 Real-time traffic monitoring dashboard  
- 🚦 Traffic signal status display  
- 🌐 Seamless integration with backend APIs  
- 📱 Responsive and user-friendly UI  

---

## 🛠️ Tech Stack

- React  
- JavaScript (ES6)  
- HTML5  
- CSS3

## ⚙️ Setup & Installation

### 1️⃣ Navigate to Frontend Folder

cd frontend
2️⃣ Install Dependencies
npm install
3️⃣ Run the Application
npm start

👉 App runs on: http://localhost:3000

🔗 Backend Integration

Ensure the backend server is running at:

http://localhost:5000

Example API call:

fetch("http://localhost:5000/api/traffic")
